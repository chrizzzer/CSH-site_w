WoLifeX

WoLifeX

Project Description

Have you ever dreamt of not only being the tourist in a foreign country but really experiencing what life there is like?

Living life from different perspectives broadens our horizon, gives us different perspectives and helps us to understand what truly matters for us in life.

WoLifeX gives you the unique opportunity to live in truly rural villages and become part of genuine community unpolluted by tourism. You have the chance to live as a ricefarmer, a monk, fisherman, or whatever you imagine in your wildest dreams.

In addition to shaping and developing your own personality, you will become yourself an opportunity and window for people from rural areas to catch a glimpse of the Western world.

Treating each other tolerantly and with respect is essential for this experience. To ensure that we have trusted contacts locally available and a continuous review system guaranteeing a safe and fruitful experience.

MotionJump

MotionJump

Project Description

MotionJump is a HTML5 game based on Phaser.io and the tracker.js library. It stands out from other games through its control by webcam-tracked motion of colors. In this example the game is controlled by any yellow item held in front of the webcam (here the yellow bracelet). Left, right or upwards moments of the iteam cause the character to respectively move in this direction or jump. The game runs on a node.js server and can be played in any common browser.

Twitterpops

Project Description

TwitterPops show up in real-time on a live view of google maps in your direct surroundings. Always stay up to date with the top trends and what's happening around you. No matter if it's a popular bar or a night club you are looking for or you wanna know the latest news or dangers in your area. This app will serve your purpose in every big city, especially in New York City!

Especially in areas like Harlem this can help you to explore new exciting activities and really get to know the suburb and its endless possibilities.

Further this app will use natural language processing to further categorize and segment tweets according to keywords and make it easier for you to discover the trending topics you like in your area.

WEventus

WEventUs

Project Description

WEventUs aims to solve the following problems:

  • The web doesn't offer a place where you can see all events in a target area displayed nicely in a list or on a map.
  • The web doesn't offer a place where you can see all events in a target area displayed nicely in a list or on a map.
  • Using Google to manually search for events is inefficient.
  • Eventful.com and Eventbrite.com are not comprehensive.

Potential: If there's a single tool that allows you to discover any event that has been posted onto the web this would be the place to advertise events and connect other services as well. The ability to grab all events via the Facebook Graph API and a custom-made RESTful API make it possible to scrape the web entirely and pull all events into one place. In addition to this, it's very easy to allow users to submit new events within a single application interface, and utilizing the backwards compatibility of the Graph API to post events to Facebook or the RESTful API to post events to a website would allow a single Event Discovery Tool to really work and make an impact.

WEventful does just this:
Scrapes Facebook for events using Graph API based on name of city and adds these events to the map based on users filtering their searches. Custom RESTful API available online is ready to tap into so that anyone with a website can POST their events through our API making them available to the Google Maps API that is running the Google Map on our site. Scrapes Eventbrite and Eventful for their listed events and pulls them in based on city name and zipcode. Clearly lists all events in a list view, or marks them in a Google Map View. Allows users to add their own events based on zipcode.
